下载和安装VSCode

  • 官网地址

Windows用户直接下载安装即可

linux用户可以选择下载对应安装包或者直接使用命令行安装 可参照这里🏃

  • Ubunut18.04安装
  • 也可以选择从 Snap Store 中安装
    • sudo snap install --classic code 安装稳定版
    • sudo snap install --classic code-insiders 安装测试版

实用插件

👇不需要配置就可以直接使用的部分插件

  • Chinese ——如果不是想锻炼自己英文能力的话还是老老实实下载一个中文插件吧

  • Bracket Pair Colorizer 2 ——彩色括号，不仅容易区分括号的层级，而且非常美观

  • Indenticator —— 直观的显示出当前的缩进

  • Code Spell Checker —— 检查英文单词拼写是否有误

👇这些插件可以通过修改 Settings.json 中的配置来达到自定义

  • Doxygen Document —— 自动生成注释文档 - 配置文档

  • TODO HighLight —— 让TODO的部分变成高亮 - 配置文档

  • Code Runner —— 运行多种语言的代码段或代码文件 - 配置文档

美化

🌸颜值=生产力

设置字体
字体选择 Source Code Pro 一款很适合编程开发的等宽字体

  • Adobe Source Code Pro Github
    • Windows 将TTF/目录下想要的字体复制到C:\Windows\Fonts\中，即可在系统字体中找到Source Code Pro
    • Ubuntu 打开OTF/目录，打开自己想要的字体直接点击安装即可

然后在 settings.json 中加入

"editor.fontFamily": "Source Code Pro, 'Source Code Pro Medium' "

然后保存修改即可修改字体

📄使用VSCode编辑markdown

  • VSCode肯定可以变成你最喜欢的 md 编辑器

Git

VSCode已经集成了 Git 的功能

Git相关插件

  • GitLens —— 文档

  • Git History —— 文档

  • gitignore —— 文档

¶在Windows中将 Git bash 集成到终端中

  • 在 settings.json 中加入 "terminal.integrated.shell.windows": "你的安装路径\\Git\\bin\\bash.exe"
  • 保存后，重新打开vscode中的集成终端，就会看到其实是打开了 Git bash
