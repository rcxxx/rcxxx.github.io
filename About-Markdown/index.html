¶一些Makrdown的编辑方法收集

¶提高效率

  • 首先你需要一个优秀的 Markdown 文档编辑器

    • 推荐 VSCode
      • VSCode配置Markdown相关插件
    • 推荐 Typora
      • 一款所见即所得的Markdown编辑器
      • Typora 完全使用详解

  • 一些生成Makrdown模板的插件的使用方法

¶提高观感

  • Markdonw 可以通过内嵌 HTML 语法来达到更好的展示效果

    • HTML 参考手册
    ¶设置文本居中，左对齐，右对齐等

    <p align="value">文本内容</p>

    value 方法
    left 左对齐内容
    right 右对齐内容
    center 居中对齐内容
    justify 对行进行伸展，每行都可以有相等的长度
    ¶修改字体、字号、颜色

    <font color="color" size=[size] face="字体">展示的内容</font>

    • <font color="blue" size=4 face="华文行楷">蓝色的华文行楷字体</font>
      • 效果：蓝色的华文行楷字体
    • 颜色值支持十六进制颜色值，也有一些预设的颜色值，具体对照可以参考这篇文章
    ¶修改背景颜色

    <span style="background-color: 'color'">展示的内容</span>

    • <span style="background-color: #D3D3D3">“浅灰色”背景</span>
      • 效果：“浅灰色”背景
    ¶实现上标、下标、下划线、上划线
    • <sup>xxx</sup> 实现上标
    • <sub>xxx</sub> 实现下标
    • <u>xxx</u> 实现下划线
    • <span style="text-decoration: overline;"> </span> 实现上划线
    ¶关于图片的处理

    Markdown ![]() 格式插入的图片默认左对齐，大小受图片实际大小影响

    • 可以用 HTML 的语法来固定图片的宽和高
      • 效果👇 <img width=100 height=100 src="图片地址" />
      • 100x100
        • 还可以通过 <center> 修饰符实现居中👇
      • 200x200
    ¶特殊字符
    • 具体可以参考——HTML 符号
    ¶数学公式
  • Markdonw 支持 Emoji😄 表情
    • 可以通过一些插件实现实时预览以及对 HTML 格式的渲染
    • 👉这个网站收集了很多的 Emoji 代码，如果有需要可以在上面寻找
