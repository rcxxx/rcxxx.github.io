¶实用插件

  • Chinese ——如果不是想锻炼自己英文能力的话还是老老实实下载一个中文插件吧
  • Bracket Pair Colorizer 2 ——彩色括号，不仅容易区分括号的层级，而且非常美观
  • Indenticator —— 直观的显示出当前的缩进
  • TODO HighLight —— 让TODO的部分变成高亮
  • Code Spell Checker —— 检查英文单词拼写是否有误

¶美化

🌸VSCode颜值越高=生产力越高

  • 字体设置

    • Source Code Pro —— 一款很适合编程开发的等宽字体
      • Adobe Source Code Pro Github
      • Windows 将TTF/目录下想要的字体复制到C:\Windows\Fonts\中，即可在系统字体中找到Source Code Pro
      • Ubuntu 打开OTF/目录，打开自己想要的字体直接点击安装即可
      • 然后在 settings.json 中加入 "editor.fontFamily": "Source Code Pro, 'Source Code Pro Medium' " 然后Ctrl + S即可修改字体

  • 颜色主题🎨

    • Material Theme 颜值很高的主题 —— 谁说程序员不浪漫

  • 文件图标主题

    • Material Theme Icons 与上面的颜色主题的风格适配的图标
    • VSCode Icons 同样推荐

  • 这里推荐一个Windows和MAC的美化插件

    • vibrancy —— 一键启用毛玻璃效果

¶Markdown

📄使用VSCode编辑markdown

  • VSCode肯定可以变成你最喜欢的 md 编辑器
  • Markdown All in One —— 编辑实时预览，方便的快捷操作
  • Markdown Preview Github Styling —— 改变Markdown预览为github主题
  • Markdown Emoji —— 可以实时预览Markdown中的emoji表情
    • 各种emjio的代码
  • Markdown Table Prettifier —— 格式化表格，更适合阅读

¶leetCode

¶Git

VSCode中如何使用Git

  • Git相关插件
    • a
  • 在Windows中将 Git bash 集成到终端中
    • 在 settings.json 中加入 "terminal.integrated.shell.windows": "你的安装路径\\Git\\bin\\bash.exe"
    • 保存后，重新打开vscode中的集成终端，就会看到其实是打开了 Git bash
