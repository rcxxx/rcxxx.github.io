Ubuntu18.04 从源码编译 OpenCV

  • OpenCV官网
  • 构建参考官网tutorial
  • 系统版本 Ubuntu18.04LTS
  • 系统硬件：
    • CPU：
    • GPU：
  • OpenCV：4.1.0
  • GCC：7.5.0
  • CMake：

¶仅编译CPU版本

  • 如果你的设备没有独立显卡，或者是不需要开启GPU加速

¶开启GPU支持

  • OpenCV支持使用GPU进行加速运算，如果你的设备使用NVIDAI的独立显卡，并且需要开启GPU支持，则可参考进行配置

¶从源码编译OpenCV——仅CPU

  • 安装 必须的包
    • libjasper-dev
¶获取 OpenCV 源码 —— 有两种方式
  • git Clone
    • 然后分别 cd 到目录中 git checkout <branches>
    • 这里如果选择通过 git 下载，编译的时候则需要注意设置 cmake 参数的路径
  • 从官网下载压缩包
    • opencv-4.1.0
    • opencv_contrib-4.1.0
    • opencv_extra-4.1.0
  • 这里选择第二种方式，下载完成后将 opencv 解压至自己的工作目录,opencv_contrib 解压至 opencv 的目录中
    • 示例
¶编译

  • 新建 build/ 目录存放编译生成的文件

  • 配置 cmake 参数，修改 testdata 和 contrib 的路径

  • 等待 makefile 生成完毕后开始编译 —— 过程保持网络通常，有一些包需要下载

    • 可以去泡壶茶然后休息一下，看看番，编译速度取决于CPU
    • 可以看看这里的视频 —— 不是广告

  • sudo make install

    • 这一步将编译好的 .so 文件以及头文件安装到 /usr/local 目录下
¶编译完成，将 OpenCV 添加到环境变量
  • sudo gedit /etc/ld.so.conf.d/opencv.conf
  • 打开 opencv.conf 在其中添加 /usr/local/lib
  • sudo gedit /etc/bash.bashrc
  • 打开 bash.bashrc 在最后面添加
  • sudo ldconfig
¶测试
  • 新建一个终端
    • 当前目录下会生成一个 example
  • 接入摄像头然后 ./example
  • 成功启动相机并显示 Hello OpenCV 字样
  • 如果你没有摄像头
  • 没有摄像头还玩什么OpenCV，洗洗睡吧
¶编译完成
¶从源码编译OpenCV——开启GPU加速
  • 获取源码
¶安装所需依赖
  • libjasper-dev
  • 开启GPU支持还需要安装这些
¶编译
  • 新建 build/ 目录存放编译生成的文件
  • 配置 cmake 参数，修改 testdata 和 contrib 的路径
  • 等待 makefile 生成完毕后开始编译 —— 过程保持网络通常，有一些包需要下载
    • 这里编译时间更久，你可以叫几个朋友一起喝茶 (= =!)
  • sudo make install 编译完成后安装
